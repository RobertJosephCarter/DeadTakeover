# DeadTakeover

DeadTakeover is a browser-based open-world zombie survival FPS built with Three.js and Vite.

## Live Demo

- Play now: `https://xrctz.github.io/ai-game-lab/games/zombie/`

## Repository Layout

This repository stores the main game source in:

- `Zombie Open World Game/`

## Quick Start (Local)

```bash
cd "Zombie Open World Game"
npm install
npm run dev
```

Then open the local Vite URL shown in your terminal (usually `http://localhost:5173`).

## What To Expect

- Multiple playable maps with distinct atmosphere and pacing
- AI teammates and special infected archetypes
- Barricade building, scavenging, and survival progression systems
- Browser-first gameplay with Three.js rendering
